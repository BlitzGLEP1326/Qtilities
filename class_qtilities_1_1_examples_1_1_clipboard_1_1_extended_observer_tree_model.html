Detailed Description

The ExtendedObserverTreeModel class is an example of how Qtilities::CoreGui::ObserverTableModel can be subclassed.

The ExtendedObserverTreeModel class is an example of how Qtilities::CoreGui::ObserverTreeModel can be subclassed to add additional functionality to your model and to Qtilities::CoreGui::ObserverWidget widgets.

This example will add additional columns.
