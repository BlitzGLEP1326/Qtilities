Modules Overview

%Qtilities consists out of the following modules:
Logging Module: Provides an advanced logging solution.
To add the correct paths to your include paths and link to the correct libraries, add this in your .pro %file: \code QTILITIES += logging; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module, add this to your .h/.cpp files: \code #include // In your .cpp files, use this namespace: using namespace QtillitiesLogging; \endcode
Core Module: Provides the core %Qtilities classes.
To add the correct paths to your include paths and link to the correct libraries, add this in your .pro %file: \code QTILITIES += core; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module, add this to your .h/.cpp files: \code #include // In your .cpp files, use this namespace: using namespace QtillitiesCore; \endcode
CoreGui Module: Provides the core %Qtilities GUI classes.
To add the correct paths to your include paths and link to the correct libraries, add this in your .pro %file: \code QTILITIES += coregui; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module, add this to your .h/.cpp files: \code #include // In your .cpp files, use this namespace: using namespace QtillitiesCoreGui; \endcode
Project Management Module: Provides project management classes.
To add the correct paths to your include paths and link to the correct libraries, add this in your .pro %file: \code QTILITIES += project_management; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module, add this to your .h/.cpp files: \code #include // In your .cpp files, use this namespace: using namespace QtillitiesProjectManagement; \endcode
Extension System Module: Provides an extension system based on the Qt plugin architecture.
To add the correct paths to your include paths and link to the correct libraries, add this in your .pro %file: \code QTILITIES += core; include(Qtilities_Path/src/Qtilities.pri) \endcode To include all files in this module, add this to your .h/.cpp files: \code #include // In your .cpp files, use this namespace: using namespace QtillitiesExtensionSystem; \endcode
