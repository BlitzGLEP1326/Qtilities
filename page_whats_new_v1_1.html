What's New In Qtilities v1.1

This page and overview of new classes, and functions added to existing classes in the v1.1 release.

A detailed overview of changes and bug fixes can be found in the detailed change log. The change log uses the following legend to list items.

  • [+] New feature added
  • [*] Bug fix
  • [#] Change/improvement to old code

For comment, queries or bug reports please get in touch.

  • Support & Mailing List: http://groups.google.com/group/qtilities?hl=en?hl=en/
  • Issue List: https://github.com/JPNaude/Qtilities/issues

New Classes

Qtilities v1.1 provides the following new classes:

  • Qtilities::CoreGui::HelpManager
  • Qtilities::CoreGui::GroupedConfigPage
  • Qtilities::CoreGui::Interfaces::IGroupedConfigPageInfoProvider
  • Qtilities::ProjectManagement::ProjectsBrowser

The help manager class is complimented through a Help plugin which contains a set of classes, all grouped under the Qtilities::Plugins::Help namespace.
