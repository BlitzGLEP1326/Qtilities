What's New In Qtilities v1.4

This page and overview of new classes, and functions added to existing classes in the v1.4 release. This release contains a handful of code changes, and the main reason for the release is to add LGPL v2.1 to the list of licenses under which Qtilities is available. The project also underwent some additional changes, described below.

A detailed overview of changes and bug fixes can be found in the detailed change log.

For comment, queries or bug reports please get in touch.

  • Support & Mailing List: http://groups.google.com/group/qtilities?hl=en?hl=en/
  • Issue List: https://github.com/JPNaude/Qtilities/issues

General Changes

  • Qtilities is now also available under LGPL v2.1.
  • Added a new wiki for the Qtilities project at https://github.com/JPNaude/Qtilities/wiki.
  • The Qtilities website moved from http://www.qtilities.org to http://jpnaude.github.io/Qtilities/.
  • suppo.nosp@m.rt@q.nosp@m.tilit.nosp@m.ies..nosp@m.org is no longer available, please use the mailing list for support from here on.
  • The Qtilities blog is no longer available.

New Classes & Structs

Qtilities v1.4 provides no new classes or structs.

  • Qtilities::Core::Interfaces::ITask::setLastErrorMessagesStackSize()
